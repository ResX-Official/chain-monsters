# chain-monsters
the solidity code for a onchain game 

// Features:
// - Mint new monsters with random attributes (using blockhash for pseudo-RNG).
// - Breed two monsters to create a new one with inherited traits.
// - Battle monsters: Simple turn-based logic resolved on-chain.
// - Level up monsters by winning battles.
// - Marketplace for buying/selling monsters.
// - Events for all actions.
// - Modifiers for ownership and validation.
